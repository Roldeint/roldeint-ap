Proyectos Realizados

1. Automatización del ciclo de vida de usuarios

Altas, modificaciones y bajas de cuentas corporativas

Objetivo

Optimizar el proceso de creación, modificación y terminación de cuentas de usuario en plataformas corporativas, reduciendo tareas manuales y fortaleciendo el control de accesos.

Alcance

  • Integración entre ServiceDesk Plus, ADManager Plus y Microsoft 365.
  • Automatización de altas, cambios y bajas de usuarios.
  • Asignación de grupos, permisos y fechas de expiración.
  • Generación de logs, trazabilidad y soporte a auditoría.

Tecnologías utilizadas

Python JavaScript ServiceDesk Plus ADManager Plus Microsoft Graph API Active Directory Microsoft 365

Resultados

  • Reducción de tiempos operativos.
  • Mayor trazabilidad y control del proceso.
  • Disminución de errores manuales.
  • Fortalecimiento del control de identidades y accesos.

Optimización de captura de información para procesos TI

Objetivo

Mejorar la experiencia de usuario y la calidad de captura de información en formularios de solicitudes TI, facilitando la automatización de procesos posteriores.

Alcance

  • Automatización de dependencias entre ubicación, contrato, servicios y cuentas.
  • Validación y estandarización de datos antes del procesamiento.
  • Mejora de formularios usados en flujos corporativos críticos.

Tecnologías utilizadas

JavaScript React UI ServiceDesk Plus Select2 JSON

Resultados

  • Formularios más intuitivos y controlados.
  • Menor reproceso por errores de captura.
  • Mejor calidad de la información para automatización posterior.
  • Mayor alineación entre negocio y operación TI.

4. Control de acceso a la red (NAC)

Fortalecimiento de seguridad y cumplimiento

Objetivo

Fortalecer la seguridad de la red corporativa restringiendo el acceso de dispositivos externos y alineando la operación con controles de seguridad y cumplimiento.

Alcance

  • Participación en el análisis e implementación de una solución NAC.
  • Evaluación de riesgos y brechas de seguridad.
  • Apoyo al cumplimiento de lineamientos asociados a ISO 27001:2022.
  • Revisión de impacto sobre operación e infraestructura.

Tecnologías y enfoques

NAC Control de acceso Gestión de riesgos Cumplimiento Infraestructura TI

Resultados

  • Mayor visibilidad sobre acceso a red.
  • Identificación de brechas en la implementación.
  • Fortalecimiento del enfoque preventivo de seguridad.
  • Aporte a iniciativas de cumplimiento y mejora continua.
